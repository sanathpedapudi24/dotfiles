## Deploying to Vercel (frontend)

The repo is structured as a monorepo with the React app in `/frontend` and the Node.js API in `/backend`.

1) **Install**: `npm install` (root) then `cd frontend && npm install --legacy-peer-deps`.
2) **Build**: `cd frontend && npm run build` (uses CRACO).
3) **Vercel settings** (recommended):
  - Project root: repo root
  - Install command: `cd frontend && npm install --legacy-peer-deps`
  - Build command: `cd frontend && npm run build`
  - Output directory: `frontend/build`
4) **API**: The Express API under `/backend` is not deployed on Vercel; host it separately (e.g., Render/Fly/railway) and point the frontend to that base URL via environment/config.
